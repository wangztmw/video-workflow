# 《墨剑》— 完整故事分镜脚本

## 故事概要
少年剑客在荒废古寺中发现一幅活着的古画，画中墨迹化为人形与之对战。他必须领悟"以剑为笔、以意为墨"的境界，才能收服画中灵。

## 角色设定
- 少年剑客：18岁，黑衣束发，面容清秀，手持无锋铁剑
- 墨影：画中墨水形成的人形，半透明，边缘有墨迹流动

## 分镜列表（10段，每段约10-15秒）

### Segment 0: 古寺寻卷
- 时长: 12s (289帧@24fps)
- 场景: 荒废古寺内部，阳光从破窗洒入，尘埃飞舞
- 画面: 少年推开腐朽木门，走进布满灰尘的大殿，目光落在一卷泛黄画轴上
- 转场: 镜头缓缓推进画轴，画面充满整个镜头

### Segment 1: 墨卷展开
- 时长: 12s
- 场景: 古寺内，少年面前
- 画面: 少年解开丝绳，画卷徐徐展开，露出水墨山水。画面中的墨迹开始微微颤动
- 转场: 镜头推进画卷中的一滴墨，墨滴占据整个画面

### Segment 2: 墨影成形
- 时长: 12s
- 场景: 古寺大殿
- 画面: 墨滴从画中浮起，在空中旋转、膨胀，逐渐凝聚成一个人形轮廓——墨影诞生
- 转场: 墨影完全成形后，镜头快速后拉展示双方对峙

### Segment 3: 初次交锋
- 时长: 12s
- 场景: 古寺大殿，尘土飞扬
- 画面: 墨影率先出击，手臂化为墨鞭抽向少年，少年拔剑格挡，金属与墨碰撞溅出墨花
- 转场: 少年被震退数步，镜头跟随他撞向殿内石柱

### Segment 4: 节节败退
- 时长: 12s
- 场景: 古寺大殿
- 画面: 墨影攻势如潮，少年左支右绌，剑法虽精却无法伤及墨影分毫。墨影化出数条墨龙环绕
- 转场: 少年被击倒在地，剑脱手滑出。镜头俯拍少年的脸，背后是逼近的墨影

### Segment 5: 师父遗音
- 时长: 12s
- 场景: 少年的回忆空间（色调变为暖黄）
- 画面: 少年闭眼，脑海中浮现师父的身影。师父以树枝代剑，在地面描绘墨竹，口中念着口诀
- 转场: 树枝划过地面的墨迹与现实中剑的轨迹重叠，镜头回到现实

### Segment 6: 以剑为笔
- 时长: 12s
- 场景: 古寺大殿
- 画面: 少年睁开眼，目光不同了。他捡起剑，但握法变了——如握毛笔。剑尖在地面划过，留下墨痕
- 转场: 他缓缓起身，剑尖指向墨影，剑身开始发出淡蓝光芒

### Segment 7: 墨舞对决
- 时长: 12s
- 场景: 古寺大殿
- 画面: 少年主动出击，每一剑挥出都在空中留下墨迹——不是砍，是写。墨影的攻击被他的"笔画"化解
- 转场: 少年在空中写出一个巨大的"定"字，墨影被定住。镜头环绕两人旋转

### Segment 8: 收墨归卷
- 时长: 12s
- 场景: 古寺大殿
- 画面: 少年以剑尖轻点墨影眉心，墨影从固态崩解为漫天墨滴，如雨般落回画卷中。画卷上出现少年的画像
- 转场: 镜头推进画卷上的少年画像，画中的他嘴角似乎带着微笑

### Segment 9: 破晓而出
- 时长: 12s
- 场景: 古寺门口，天边微亮
- 画面: 少年走出寺门，晨曦映照他的脸。他回望一眼古寺，将画卷收入怀中，踏上石阶山道。远方山峦叠翠
- 转场: 镜头缓缓升空，俯瞰整个山谷古寺，少年身影渐小

## 视觉风格
- 水墨画风（ink wash painting / sumi-e style）
- 色调：古寺内为灰蓝冷色调，回忆为暖黄，决战为黑白对比
- 摄像机：多用缓慢推进(push in)、环绕(orbit)、仰拍(low angle)
